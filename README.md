# Airdrop Advanced Tool

Send any ERC20 tokens to thousands of receivers

## Installation 

`yarn`

## Configuration

The configuration is specified in the file `.env`

## System start

### Production

  - build: `yarn build`

  - start: `pm2 start pm2/script_airdrop.sh`

  - Listenning port: `3000`

### Local development

  - build: `yarn build`

  - start: `yarn start`

  - Listenning port: `3000`
